# FastAPI App

## Prerequis:

```bash
sudo apt install python3 python3-pip
pip install fastapi uvicorn python-jose pydantic
```

## Run the app with "uvicorn":
```bash
uvicorn api_fastapi_v5:app --reload
```

# Base api location:
http://127.0.0.1:8000

# Swagger api location:
http://127.0.0.1:8000/docs#/
